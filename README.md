# 🎮 2D Maze Game (Java)

An interactive jungle-themed maze game built using Java and Object-Oriented Programming concepts. The game features a GUI interface and intelligent enemy behavior using pathfinding algorithms.

---

## 🚀 Features
- 🧠 Pathfinding using BFS / A*
- 🎯 Enemy AI (Tiger chasing logic)
- 🖥️ GUI built with Java Swing
- 🎮 Multiple game modes
- 💀 Game over & restart functionality

---

## 🛠️ Tech Stack
- Java
- OOP (Object-Oriented Programming)
- Java Swing (GUI)

---

## 🧩 Algorithm
The game uses pathfinding algorithms like BFS/A* to determine the shortest path for enemy movement, making gameplay more dynamic and challenging.

---

## ▶️ How to Run
1. Clone the repository  
2. Open in any Java IDE (VS Code / IntelliJ)  
3. Run the main class
